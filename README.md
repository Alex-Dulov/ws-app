# ws-app
simple chat

-----------------------------------------------------------------
Установка
-----------------------------------------------------------------
git clone ...  
nvm use 14.20.1  
yarn  
yarn start  

-----------------------------------------------------------------
Краткое описание
-----------------------------------------------------------------

На данный момент реализовано:
- Переписка в общем чате
- Возможность размещать посты (будут видны во вкладке Новости)
- Настройки, модалка при входе, список пользователей и проверка id пока не реализована
